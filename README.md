# Excel处理工具

一个功能强大的Excel数据处理工具，支持多种字段处理规则，可视化配置，批量处理数据，以及Excel与JSON之间的相互转换。

## 主要功能

### 1. 智能输出（一个 Excel 按规则生成另一个Excel）

- **多种数据处理规则**：支持直接复制、前缀添加、后缀添加、前后添加、固定值、正则替换等多种数据处理规则
- **可视化配置**：通过直观的界面配置处理规则，支持拖拽操作
- **配置管理**：支持保存、加载配置，方便重复使用
- **自动修复**：自动检测和修复Excel文件的样式问题

### 2. 表格切分（一个Excel按规则生成多个Excel）

- **多列切分**：支持按多个列进行表格切分
- **保留表头**：可指定保留的表头行数
- **自定义输出目录**：支持指定切分后的文件保存目录
- **批量处理**：支持大批量数据的切分处理
- **进度显示**：实时显示切分进度和状态

### 3. 转为JSON

- **灵活列选择**：可选择需要转换的列
- **类型转换**：支持将列数据转换为字符串、数字或时间戳类型
- **字段重命名**：可自定义输出JSON中的字段名
- **自动生成文件名**：默认使用工作表名作为JSON文件名

## 使用步骤

### 使用方法

1. **安装依赖**：

   ```
   pip install -r requirements.txt
   ```
2. **运行程序**：

   ```
   python main_qt6.py
   ```

   或双击 `run_qt6.bat` 运行
3. **智能输出操作**：

   - 点击"打开Excel文件"按钮选择要处理的文件
   - 从下拉菜单中选择要处理的Sheet
   - 查看左侧显示的原始数据列信息和预览
   - 点击"新建"按钮创建新配置
   - 选择输入列（可选，固定值规则不需要）
   - 输入新列名
   - 选择处理规则并填写规则参数
   - 点击"添加到配置"按钮将配置添加到列表
   - 点击"生成Excel"按钮处理并保存结果文件
4. **表格切分操作**：

   - 打开Excel文件并选择工作表
   - 在"表格切分"标签页中设置保留行数
   - 从左侧列信息列表拖动列到"拆分列"列表
   - 调整拆分列的顺序（上下移动）
   - 选择输出目录（可选）
   - 点击"执行切分"按钮开始切分
5. **转为JSON操作**：

   - 打开Excel文件并选择工作表
   - 在"转为json"标签页中选择需要转换的列
   - 为每列设置类型（字符串、数字、时间戳）
   - 自定义输出字段名（可选）
   - 选择输出文件路径（可选）
   - 点击"转换为JSON"按钮开始转换

### 界面说明

### 左侧区域 - 原始数据

- **列列表**：显示所有可用列及示例数据，支持拖拽操作
- **数据预览**：显示原始数据的前10行，方便查看数据格式

### 右侧区域 - 功能标签页

#### 智能输出标签页

- **配置列表**：显示所有已配置的处理规则
- **配置表单**：用于创建和编辑处理规则
- **操作按钮**：快速保存、删除、清空、生成Excel

#### 表格切分标签页

- **切分配置**：设置保留行数、拆分列、输出目录
- **操作按钮**：帮助、执行切分、取消
- **进度显示**：显示切分进度和状态

#### 转为JSON标签页

- **转换配置**：设置输出文件路径
- **列选择**：选择需要转换的列，设置类型和输出字段名
- **操作按钮**：转换为JSON
