# Excel处理工具

一个功能强大的Excel数据处理工具，支持多种字段处理规则，可视化配置，批量处理数据。

## 版本说明

本项目提供两个版本：
- **tkinter版本**：`main.py` - 基于Python内置tkinter库，无需额外安装UI框架
- **Qt6版本**：`main_qt6.py` - 基于PyQt6框架，提供更现代化的界面体验

## 功能特点

- 支持多种数据处理规则
- 可视化配置界面
- 实时预览和编辑
- 只输出配置的字段
- 支持固定值生成新列

## 处理规则

### 1. 直接复制
将原始列的数据直接复制到新列，不做任何修改。

**参数**：无

### 2. 前缀添加
在原始数据的前面添加指定的文本。

**参数**：
- 前缀：要添加的文本内容

**示例**：
- 原始数据：`123`
- 前缀：`ID_`
- 结果：`ID_123`

### 3. 后缀添加
在原始数据的后面添加指定的文本。

**参数**：
- 后缀：要添加的文本内容

**示例**：
- 原始数据：`产品A`
- 后缀：`_v1.0`
- 结果：`产品A_v1.0`

### 4. 前后添加
在原始数据的前后同时添加指定的文本。

**参数**：
- 前缀：要添加的文本内容
- 后缀：要添加的文本内容

**示例**：
- 原始数据：`用户`
- 前缀：`ID_`
- 后缀：`_active`
- 结果：`ID_用户_active`

### 5. 固定值
忽略原始数据，为新列设置固定值。

**参数**：
- 固定值：要设置的固定文本内容

**示例**：
- 原始数据：任意值
- 固定值：`状态：激活`
- 结果：所有行的该列值均为"状态：激活"

## 使用步骤

1. **选择版本**：根据需求选择tkinter版本或Qt6版本
2. **安装依赖**：
   - tkinter版本：`pip install -r requirements.txt`
  ：`pip install -r requirements.txt`
3. **运行程序**：
   - tkinter版本：`python main.py`
  ：`python main_qt6.py`
4. **打开Excel文件**：点击"打开Excel文件"按钮选择要处理的文件
5. **选择工作表**：从下拉菜单中选择要处理的Sheet
6. **查看原始数据**：左侧显示原始数据的列信息和预览
7. **创建配置**：
   - 点击"新建"按钮创建新配置
   - 或先选择列，再点击"新建"基于该列创建配置
8. **配置处理规则**：
   - 选择输入列（可选，固定值规则不需要）
   - 输入新列名
   - 选择处理规则
   - 填写规则参数
9. **添加配置**：点击"添加到配置"按钮将配置添加到列表
10. **编辑配置**：点击列表中的配置项进行编辑
11. **生成Excel**：点击"生成Excel"按钮处理并保存结果文件

## 配置管理

- **保存配置**：将当前配置保存为JSON文件
- **加载配置**：从JSON文件加载之前保存的配置
- **清空配置**：清除所有配置项
- **删除配置**：删除选中的配置项

## 界面说明

### 左侧区域 - 原始数据
- **列列表**：显示所有可用列及示例数据
- **数据预览**：显示原始数据的前10行

### 右侧区域 - 输出配置
- **配置列表**：显示所有已配置的处理规则
- **添加配置**：用于创建和编辑处理规则
- **操作按钮**：新建、添加、删除、清空、生成Excel
